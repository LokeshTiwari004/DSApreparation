## User's Approach
`while True` loop with adjacent-guard (same pattern as #704/#35/#153). Three main branches:
1. **Left half rotated** (`nums[lower] > nums[cursor]`): check if target lies in `(nums[cursor], nums[lower])` → search left, else right
2. **Right half rotated** (`nums[cursor] > nums[upper]`): check if target lies in `(nums[upper], nums[cursor])` → search right, else left
3. **Normal sorted** (no rotation in range): standard directional search

Also checks `nums[lower] == target` and `nums[upper] == target` redundantly at each iteration.

## Pattern Analysis
**Correct pattern: Binary Search in Rotated Array.**
This extends #153's rotation detection: once you identify which half is rotated and which is sorted, you decide which half contains the target based on whether target falls within the sorted half's range.

The standard template avoids the `nums[lower] == target` / `nums[upper] == target` redundancy by using a cleaner single-`mid` comparison:

```python
def search(nums, target):
    lo, hi = 0, len(nums) - 1
    while lo <= hi:
        mid = lo + (hi - lo) // 2
        if nums[mid] == target:
            return mid
        if nums[lo] <= nums[mid]:           # left half is sorted
            if nums[lo] <= target < nums[mid]:
                hi = mid - 1
            else:
                lo = mid + 1
        else:                                 # right half is sorted
            if nums[mid] < target <= nums[hi]:
                lo = mid + 1
            else:
                hi = mid - 1
    return -1
```
Key insight: `nums[lo] <= nums[mid]` = left half is contiguous (sorted). This is cleaner than checking `nums[lower] > nums[cursor]`.

## Key Takeaways
- Use `nums[lo] <= nums[mid]` to detect sorted half — simpler than range comparisons
- Always reduce to: "one half is sorted, find if target belongs there; use BS on the other half otherwise"
- Only need to check `nums[mid] == target` once per iteration — no need for `nums[lo]` / `nums[hi]` checks
- The standard template for rotated array is about 10 lines — aim for that
