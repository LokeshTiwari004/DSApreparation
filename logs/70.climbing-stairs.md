# Problem: Climbing Stairs (#70)

**Difficulty:** Easy  
**Link:** https://leetcode.com/problems/climbing-stairs/  
**Date Solved:** 05/07/26 (v1: combinatorics), 06/07/26 (redo: iterative DP)  
**Phase:** Phase 3 – Mixed Set (Diverse)

---

## User's Approach (v1 — Combinatorics)

Combinatorial sum: Σ C(n−k, k) for k = 0 to floor(n/2). Custom `ncr(n, r)` loop. O(n²) time. TLE on large n.

## User's Approach (v2 — Iterative DP)

Iterative Fibonacci:
- Base: `a, b = 1, 2` for n=1, n=2
- Loop n-2 times: `a, b = b, a + b`
- Return `b`

---

## Correctness

✅ **Correct** — `ways(n) = ways(n-1) + ways(n-2)` is the Fibonacci recurrence.
- n=1: returns 1
- n=2: returns 2
- n=3: a=2,b=3 → 3
- n=5: a=5,b=8 → 8

---

## Pattern Analysis

**Pattern:** DP (Fibonacci) — **Correct**

**Why DP applies:**
- Optimal substructure: ways to reach step n = ways to reach n-1 (take 1 step) + ways to reach n-2 (take 2 steps)
- Overlapping subproblems: f(n) depends on f(n-1) and f(n-2)
- Space-optimized DP uses just two variables
